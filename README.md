# AI Impact on Student Performance

A data analytics project that explores how Generative AI usage relates to students' academic performance, wellbeing, and learning habits using Python and exploratory data analysis (EDA).

---

## 📖 Project Overview

Generative AI has become an increasingly common learning tool among students, raising questions about its impact on academic performance and overall wellbeing. This project analyzes a survey-based dataset to uncover patterns between AI usage, GPA, burnout risk, skill retention, and study behaviors.

The project follows a complete data analytics workflow, including data cleaning, exploratory data analysis, visualization, and interpretation of findings.

---

## 🎯 Objectives

- Clean and prepare the dataset for analysis.
- Explore patterns in student AI usage.
- Analyze relationships between AI usage and academic performance.
- Examine burnout risk and skill retention.
- Present findings through clear visualizations and insights.

---

## 📂 Dataset

The dataset used in this project was sourced from **Kaggle** and is intended for educational and analytical purposes.

**Note:** The dataset was **not created by me**. Full credit belongs to the original dataset creator on Kaggle.

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

---

## 📊 Analysis Performed

The project includes:

- Data cleaning and preprocessing
- Missing value and duplicate checks
- Exploratory Data Analysis (EDA)
- Summary statistics
- GPA comparison (Pre-Semester vs. Post-Semester)
- Weekly Generative AI usage analysis
- Burnout risk analysis
- Skill retention analysis
- Comparative analysis across majors and AI use cases
- Data visualization and interpretation

---

## 📈 Key Insights

Some of the insights explored include:

- Trends in students' GPA before and after AI usage.
- Patterns in weekly Generative AI usage.
- Relationship between AI usage and burnout risk.
- Skill retention across different AI use cases.
- Differences in academic outcomes across various majors.
